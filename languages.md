# Looping

JavaScript
```
var words = ['one', 'two', 'three', four', 'five'];
var i;
for (i = 0; i < words.length; i++) {
  document.write(words[i]);
}
```

Python
```
words = ['one', 'two', 'three', four', 'five']
for i in words:
   print(i)
```

PHP
```
$words = array('one', 'two', 'three', four', 'five');
foreach($words as $word) {
  print $word;
}
```
